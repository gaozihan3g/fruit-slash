02 / UNDER THE HOOD

没有素材包，
每一帧都由代码绘制。

水果、阴影、刀光、粒子与 UI 全部使用 Pygame 的绘图 API 生成。项目体积轻，结构直接，很适合用来学习小游戏中的物理、碰撞和状态管理。

  • 01Vector2 运动重力、速度与抛物线
  • 02线段圆碰撞高速挥刀也不漏判
  • 03粒子系统果汁、爆炸与衰减动画

def segment_circle_hit(a, b, center, radius):
    ab = b - a
    if ab.length_squared() == 0:
        return center.distance_to(a) <= radius

    t = clamp(
        (center - a).dot(ab) / ab.length_squared(),
        0, 1
    )
    return center.distance_to(a + ab * t) <= radius
